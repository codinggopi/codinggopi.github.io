# Gopinath G - Developer Portfolio

Welcome to the source code of my personal portfolio website! This repository contains the code for my dynamic, responsive, and performance-optimized portfolio, designed to showcase my projects, skills, and experience as a Full Stack Developer.

## 🚀 Live Demo
[Visit my Portfolio](https://codinggopi.github.io)

## 🛠️ Built With
- **HTML5 & CSS3** - For semantic structure and custom, rich styling (Glassmorphism, Conic Gradients).
- **Vanilla JavaScript** - For interactive logic and DOM manipulation.
- **GSAP (GreenSock)** - For smooth scroll animations, typewriter effects, and dynamic element reveals.
- **FormSubmit** - For handling the contact form seamlessly without a custom backend.
- **FontAwesome & Google Fonts** - For typography (Syne & DM Sans) and scalable vector icons.

## ✨ Features
- **Highly Optimized Performance:** GPU-accelerated animations for smooth scrolling and rendering on all devices.
- **Responsive Design:** Flawless layouts from mobile screens to desktop browsers using modern CSS techniques (dvh, Grid, Flexbox).
- **Interactive UI:** Ambient background blobs, rotating gradient borders, and smooth hover micro-interactions.
- **Accessibility & Preferences:** Built-in support for users who prefer reduced motion.
